# Overview

Level 1 evaluation, or **Model evaluation**, is the foundational stage for assessing any Generative AI application in the development sector. It focuses squarely on the technical performance of the underlying AI system, including the foundation model, prompts, and knowledge bases.&#x20;

This evaluation is critical because if the GenAI system fails to perform its core function—such as providing accurate translations or relevant, safe responses—users are unlikely to engage with the product or achieve any intended development outcomes.&#x20;

The primary objective is to track and ensure the continuous, reliable behavior of the AI over time, which often involves setting up automated and human-led assessments to measure key metrics like accuracy, response completeness, and latency.&#x20;

The insights gained at this level, often led by AI Engineers and ML Researchers, inform the necessary technical iterations to stabilize the product for higher-level evaluations.
